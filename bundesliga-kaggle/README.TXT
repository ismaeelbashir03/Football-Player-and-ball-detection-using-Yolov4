- download the yolov4.weights file from: https://drive.google.com/file/d/1YxhuhZwbRhon669bZ1FMK_5WGyQQrAK4/view?usp=sharing
- put this file in the yolov4 folder
- create a 'frames' folder in the main directory 
- put your videos in the 'videos' folder
- RUN 'python main.py' to run the program
- after video has been processed, enter y in the command line to play the images
- if you run this again after images have been processed, you can just play the images
- to process images again, delete the contents of the 'frames' folder, 
  DO NOT DELETE THE 'frames' FOLDER
